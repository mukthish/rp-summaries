The provided text introduces **PATTON**, a novel pretraining framework designed to continuously pretrain language models on **text-rich networks**, which are networks where text documents (nodes) are interconnected by semantic links (edges), such as academic papers linked by citations or e-commerce items linked by co-views.

**Core Problem Addressed:** Existing pretraining methods typically fall into two disjointed categories: traditional language models (like BERT) focus exclusively on modeling text and **ignore inter-document network structures**, while graph-based pretraining methods focus on structural topology but **struggle to capture rich textual semantics**. PATTON aims to bridge this gap by extracting contextualized semantic information from the network structure at both the token and document levels.

**Key Innovations of PATTON:** The framework utilizes a **GNN-nested Transformer architecture**, which actively inserts Graph Neural Network (GNN) modules between standard Transformer layers to allow nodes to aggregate hidden states from their network neighbors. To optimize this architecture, PATTON employs two novel joint pretraining strategies:

- **Network-contextualized Masked Language Modeling (NMLM):** This strategy focuses on token-level semantics. It extends standard masked language modeling by masking tokens within a node and forcing the model to predict them using **both the text inside the given node and the text from its neighboring nodes** in the network.
- **Masked Node Prediction (MNP):** This strategy focuses on document-level semantics by randomly masking entire nodes and training the model to correctly identify them based purely on the textual information of their adjacent neighbors. Because calculating representations for all candidate nodes is computationally expensive, the authors mathematically simplified the MNP objective into a much more efficient **pairwise link prediction task**.

**Key Findings and Performance:**

- **Consistent Superiority Across Tasks:** PATTON was evaluated on five datasets spanning academic and e-commerce domains. It significantly and consistently outperformed baseline models—including pure text pretraining methods (like BERT and SciBERT) and pure graph pretraining methods—across four diverse downstream tasks: **node classification, fine-grained category retrieval, reranking, and link prediction**.
- **Robust Knowledge Transfer:** The framework demonstrated an exceptional ability to generalize its structural learning. In the link prediction task, PATTON accurately predicted new types of semantic edges (e.g., predicting "author overlap" or "co-purchased" links) despite only being pretrained on entirely different edge types (like "citations" or "co-viewed" links).
- **Attention Map Activation:** Analysis revealed that PATTON's pretraining actively alters how the language model processes data. After pretraining, the model's attention mechanisms for "neighbor virtual tokens" became significantly more activated, meaning **the model actively learned to extract and utilize useful information from network neighbors** to enhance its understanding of the center node's text.
- **Scalability:** Combining both the NMLM and MNP strategies into a joint pretraining objective achieved superior downstream performance without adding significant time or memory complexity overhead compared to using NMLM alone.
